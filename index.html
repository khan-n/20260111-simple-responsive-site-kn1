Flexbox Layout - Part 1

To display the BOXES in a line - using (display:flex) in container.
Note that By default the "justify-content" starts from the LEFT (flex-start) side of the container area/border. To change that, we can indicate the changes as needed.
Also note that the (justify-content) works horizontally!
(example: flex-end | center | space-between | space-around)
