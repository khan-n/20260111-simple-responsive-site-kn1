Flexbox Layout - Part 2

To display the BOXES in a line - using (display:flex) in container.
Note that the (flex-direction) in default is unidirectional! To indicate the changes we can use (column | column reverse | row-reverse).
Notice that using "reverse" makes the order of the items to become reversed (123 will become 321).
